\begin{abstract}
Se analizaron diferencias de precios en leche, aceite y arroz entre Bodega Aurrera, Walmart, La Comer y Soriana mediante ANOVA, observaciones pareadas y razón de varianzas sobre datos obtenidos por web scraping. Bodega Aurrera presentó los precios más bajos en marcas masivas de leche, mientras que Walmart resultó entre \$1.65 y \$2.51~MXN más caro para productos equivalentes. Además, el arroz mostró heterocedasticidad entre cadenas.
\end{abstract}

\maketitle  

\section{Introducción}

 El presente trabajo analiza si existen diferencias estadísticamente significativas en el precio de productos de la canasta básica —leche, aceite y huevo— entre cadenas de supermercados en México. Se aplicaron tres pruebas de inferencia estadística: ANOVA de un factor, observaciones pareadas y razón de varianzas.

\section{Planteamiento del Problema}

En la actualidad, el consumidor mexicano enfrenta constantes fluctuaciones en los precios de la canasta básica debido a la inflación y las dinámicas del mercado. Frecuentemente, se asume de manera empírica que ciertas tiendas son “más baratas” o que los productos cuestan lo mismo independientemente del supermercado, especialmente cuando pertenecen a la misma cadena corporativa.

Sin embargo, existe una asimetría de información que dificulta optimizar el gasto familiar. No está estadísticamente comprobado si cambiar de supermercado genera un ahorro real y significativo en los mismos productos, o si las diferencias observadas responden únicamente al azar y al ruido del mercado.

¿Existe una diferencia estadísticamente significativa en el precio de productos de la canasta básica —leche, aceite y arroz— dependiendo exclusivamente de la cadena de supermercado donde se adquieren?

Para responder esta pregunta, se plantearon tres pares de hipótesis sobre una base de datos obtenida autónomamente de cuatro cadenas de supermercados: Bodega Aurrera, Walmart, La Comer y Soriana. Se analizaron únicamente tres productos: leche, aceite y arroz.

Para el caso de las leches se aplicó \textbf{ANOVA} de un solo factor  (Walpole et al., 2012):
\begin{itemize}
	\item \textit{\textbf{Hipótesis Nula (H\textsubscript{0})}: El precio promedio por litro de leche es estadísticamente igual en todas las tiendas. Las diferencias observadas son producto del azar. $\mu$Aurrera = $\mu$Walmart = $\mu$La comer.}
	\item \textit{\textbf{Hipótesis Alternativa (H\textsubscript{1})}:  Al menos un supermercado ofrece un precio promedio significativamente distinto.}	
\end{itemize}

Para comparar directamente los precios entre Walmart y Bodega Aurrera, se utilizó una prueba de hipótesis para \textbf{observaciones pareadas}.

\begin{itemize}
	\item \textit{\textbf{Hipótesis Nula (H\textsubscript{0})}: No existe ninguna diferencia real; cualquier disparidad observada en la muestra se atribuye al error de muestreo y al ruido del mercado.}
	\item \textit{\textbf{Hipótesis Alternativa (H\textsubscript{1})}:  Existe una diferencia significativa entre los precios promedio.}	
\end{itemize}
Se fijó un nivel de significancia de $\alpha$ = 0.05.

Para el caso del arroz se realizó una \textbf{prueba de razón entre varianzas} para comparar la dispersión de precios entre tiendas.

\textbf{Hipótesis para Shapiro-Wilk} (para cada tienda):
\begin{itemize}
	\item \textit{\textbf{Hipótesis Nula (H\textsubscript{0})}: Los datos siguen una distribución normal.}
	\item \textit{\textbf{Hipótesis Alternativa (H\textsubscript{1})}:  Los datos no siguen una distribución normal.}	
\end{itemize}
Si el valor p > 0.05 no se rechaza H\textsubscript{0} y se asume normalidad. Si el valor p < 0.05 se rechaza H\textsubscript{0}.

\textbf{Hipótesis para razón entre varianzas} (por cada par de tiendas):
\begin{itemize}
	\item \textit{\textbf{Hipótesis Nula (H\textsubscript{0})}: $\sigma_1^2 = \sigma_2^2$  Las varianzas de los precios entre las dos tiendas comparadas son iguales}
	\item \textit{\textbf{Hipótesis Alternativa (H\textsubscript{1})}: $\sigma_1^2 \neq \sigma_2^2$  Las varianzas de los precios son diferentes}	
\end{itemize}

\textbf{Hipótesis para prueba de Levene} (las cuatro tiendas incluyendo La Comer):
\begin{itemize}
	\item \textit{\textbf{Hipótesis Nula (H\textsubscript{0})}: Las varianzas son homogéneas (homocedasticidad).}
	\item \textit{\textbf{Hipótesis Alternativa (H\textsubscript{1})}:  Las varianzas no son homogéneas (heterocedasticidad).}	
\end{itemize}

\section{Metodología}

La base de datos fue construida mediante web scraping de las plataformas digitales de cuatro cadenas de supermercados: Walmart, Bodega Aurrera, La Comer y Soriana, durante abril de 2026, obteniendo 1,513 registros. Debido a la naturaleza automatizada de la recolección, se realizó una limpieza manual para eliminar registros no alimenticios capturados dentro de las categorías de interés. Los precios de leche se estandarizaron manualmente al precio por litro para hacer comparables presentaciones de distinto volumen. El análisis comparativo se realizó efectivamente siendo mas representativas las cadenas \textit{Walmart} y \textit{Bodega Aurrera}. 

Para el análisis de precios de leche entre supermercados se empleó \textbf{ANOVA de un factor}, prueba que permite determinar si las medias de tres o más grupos independientes son estadísticamente iguales.

En análisis comparativo de precios entre Walmart y Bodega Aurrera, la variable de interés corresponde a la diferencia entre el precio de cada producto para productos equivalentes. La hipótesis nula plantea que no existe una diferencia real en los precios promedio y que cualquier variación se debe únicamente al azar o al error de muestreo. Por otro lado, la hipótesis alternativa sostiene que sí existe una diferencia sistemática y estadísticamente significativa entre los precios de ambas tiendas. Para evaluar estas hipótesis, se estableció un nivel de significancia de $\alpha$ = 0.05, aceptando una probabilidad máxima del $5\%$ de rechazar incorrectamente la hipótesis nula cuando ésta sea verdadera. 

Para el caso del arroz se aplicó la prueba de \textbf{razón entre varianzas}, que permite comparar la dispersión de precios entre tiendas e identificar cuál presenta mayor inestabilidad. Una varianza alta indica precios dispersos; una varianza baja indica precios consistentes. Dado que esta prueba requiere normalidad poblacional, se aplicó previamente la prueba de \textbf{Shapiro-Wilk} a cada tienda para verificar este supuesto. En los casos donde no se cumplió normalidad, se empleó la prueba de \textbf{Levene} como alternativa, ya que no requiere el supuesto y puede aplicarse a más de dos grupos simultáneamente.

Para las \textbf{observaciones pareadas} se compararon 223 pares de diferencias {D\textsubscript{i}, se calcularon la media muestral $\bar{D}$  y la desviación estándar muestral S\textsubscript{D} de dichas diferencias. El error estándar de la media de las diferencias se obtiene como: $S_{\bar{D}}=\frac{S_D}{\sqrt{n}}$
	
El estadístico de prueba sigue una distribución t de Student con $\nu$ = 222 grados de libertad, y se calcula como: $T=\frac{\bar{D}}{S_{\bar{D}}}$

Dado que la hipótesis alternativa no especifica la dirección del efecto, es decir, no se afirma cuál de las dos tiendas es más cara, se realizó una prueba bilateral. El estadístico observado fue T = 9.5583, valor que indica que la media de las diferencias se encuentra a más de 9 desviaciones estándar del error por encima del cero, alejándose de la región de no rechazo. 

Con base en la distribución t con 222 grados de libertad y un nivel de confianza del 95$\%$, se obtuvo el intervalo de confianza para la diferencia media poblacional $\mu_{D}$: $IC_{95\%}(\mu_{D})=(1.6531, 2.5119)$.

Este intervalo indica que, Con un nivel de confianza del 95\%, la diferencia promedio de precios entre Walmart y Bodega Aurrera se ubica entre \$1.65 y \$2.51. Como el intervalo no incluye al cero y el valor p resultó $p \approx 0.0000$, se rechaza $H_{0}$ con $\alpha = 0.05$. Ambos criterios confirman una diferencia estadísticamente significativa entre los precios de ambas cadenas.

Los resultados muestran que Walmart cobra, en promedio, entre \$1.65 y \$2.51 pesos más que Bodega Aurrera por los mismos productos. Aunque la diferencia individual es pequeña, puede representar un gasto acumulado relevante en una despensa completa.

\textit{Error Tipo I y Tipo II:}

Por otro lado, el error Tipo II consistiría en no detectar una diferencia real entre ambas cadenas; no obstante, dado el tamaño muestral ($n = 223$) y el valor del estadístico ($T = 9.5583$), la probabilidad de este error es reducida.

\textbf{Bodega Aurrera vs. Walmart}: 
$H_{0}=\sigma^{2}_{BA}=\sigma^{2}_{W}$, 
$H_{1}=\sigma^{2}_{BA}\ne\sigma^{2}_{W}$.
Una vez obteniendo el F Calculado y comparandolo con el F-crítico, se encontró evidencia estadísticamente significativa al 95\% de confianza para concluir que las varianzas de precios entre Bodega Aurrera y Walmart son diferentes; por lo tanto, se rechaza $H_0$: $F_{calc}=2.4924>F_{crit}=2.4060$

\textbf{Bodega Aurrera vs. Soriana}:
$H_{0}=\sigma^{2}_{BA}=\sigma^{2}_{S}$, 
$H_{1}=\sigma^{2}_{BA}\ne\sigma^{2}_{S}$.
En este caso también rechazamos la hipótesis nula($H_0$), pues existe evidencia estadísticamente significativa, con un nivel de confianza del 95\% para afirmar que las varianzas son diferentes: $F_{calc}=2.5318>F_{crit}=2.390$

\textbf{Walmart vs. Soriana}
$H_{0}=\sigma^{2}_{W}=\sigma^{2}_{S}$, 
$H_{1}=\sigma^{2}_{W}\ne\sigma^{2}_{S}$.
Al comparar el $F$ calculado con el $F$ crítico, no se encontró evidencia suficiente para afirmar que las varianzas son diferentes entre Walmart y Soriana; por lo tanto, no se rechaza $H_0$ y se asume homocedasticidad: $F_{calc}=1.0158 \leq F_{crit}=2.210$

El Error Tipo I ocurre al rechazar $H_0$ siendo verdadera; en este caso, afirmar incorrectamente que las varianzas entre tiendas son distintas. Esto podría provocar decisiones de negocio erróneas basadas en diferencias inexistentes.

El Error Tipo II ocurre al no rechazar $H_0$ siendo falsa; por ejemplo, asumir que Walmart y Soriana tienen varianzas iguales cuando realmente sus precios fluctúan de manera distinta. Esto podría llevar a ignorar diferencias competitivas relevantes entre ambas tiendas.

\section{Discusión}

Con estos resultados tenemos un rechazo parcial de la hipótesis nula, ya que depende estrictamente del posicionamiento de la marca.

Para marcas de consumo masivo como (Lala, Alpura) y marcas propias (Great Value), \textbf{rechazamos H\textsubscript{0}} $p$ < 0.05; Bodega Aurrera es significativamente más barato, generando un ahorro comprobado de hasta \$4.63 MXN por litro.

Sin embargo, para marcas de percepción premium o regional (Santa Clara, San Marcos), \textbf{no existe evidencia para rechazar H\textsubscript{0}} $p$ > 0.05; la tienda respeta un precio unificado, costando estadísticamente lo mismo en ambos formatos.

El análisis se blindó contra el Error Tipo I exigiendo niveles de significancia estrictos p < 0.05, garantizando la absoluta veracidad de los ahorros reportados. Sin embargo, se asume un margen inherente de Error Tipo II en categorías con muestras reducidas, donde el modelo estadístico podría omitir diferencias de precio reales pero matemáticamente sutiles.
