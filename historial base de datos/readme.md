Contenido de las bases creadas para los diferentes propositos del proyecto
* aceite_limpio.csv | contiene solamente el producto aceite de todas las tiendas, con precios mayores a $15 y menores a $80 incluye filtro de palabras prohibidas para conservar productos relevantes
* arroz_limpio.csv | contiene solamente el producto arroz de todas las tiendas, incluye filtro de palabras prohibidas para conservar productos relevantes
* datos.json | documento json con los productos obtenidos de forma manual
* geoprecio_ML.csv | data set para usar en la aplicacion de modelos de Machine learning, contiene los productos de pastas, frijoles, leche, aceite y arroz
* geoprecio_clean.csv | data set con categorias de normalizacion como precio por unidad de media y no solo del contenido del empaque
* geoprecio_db_productos.csv | data set original obtenido del scraping
* leche_filtrada.csv | data set de leche usado para ANOVA
* marca_leche.csv | data set de leche usado para ANOVA
